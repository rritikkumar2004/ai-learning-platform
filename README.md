# 🤖 AI Powered Learning Platform

## 📌 Project Overview
AI Powered Learning Platform ek web-based application hai jo students ko smart learning experience provide karta hai. Is platform me AI ka use karke users apne doubts solve kar sakte hain, topics learn kar sakte hain aur apni learning progress track kar sakte hain.

---

## 🎯 Objectives
- Students ko personalized learning experience dena  
- AI ke through instant doubt solving provide karna  
- Easy-to-use learning interface banana  
- Self-learning ko promote karna  

---

## 🛠️ Tech Stack Used

### Frontend:
- HTML  
- CSS  
- JavaScript  

### Backend:
- Node.js  
- Express.js  

### Database:
- MongoDB (optional / future scope)

### AI Integration:
- OpenAI API / Gemini API  

---

## ✨ Features
- 🔐 User Authentication (Login/Signup) *(optional)*  
- 🤖 AI Chatbot for doubt solving  
- 📚 Learning topics / courses  
- 📝 Quiz system *(future scope)*  
- 📊 Progress tracking *(future scope)*
